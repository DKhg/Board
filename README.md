# SpringBoot와 My-SQL을 통한 게시판 구현
<br>
<br>


## 개발 환경
* Java 11
* MySQL
* MyBatis
* Spring Boot 

<br>

## 개발 기술 스택
* Thymeleaf

<br>

## 게시판 구현 기능
* CRUD 기능 [ 생성, 조회, 수정, 삭제 ]
* 조회수 카운트 기능
* 게시글 목록 Paging 처리
